# 五行人格原型 — 用户画像增强工具

> 不是"算命"，是东方版的人格心理学框架。
> 帮助AI做更深层的用户洞察，不向用户展示五行术语。

---

## 用途

当AI做用户画像分析时，用五行原型来理解：
- 这类用户的核心驱动力是什么？
- 他们的消费决策逻辑是什么？
- 什么样的内容和话术能打动他们？

---

## 金型用户（决断型 / The Executor）

**核心驱动：** 效率、结果、控制
**典型画像：** 企业高管、创业者、金融从业者
**消费特征：**
- 愿意为节省时间付高价
- 对数据和ROI敏感
- 厌恶不确定性，喜欢明确承诺
- 决策快，但要求结果兑现

**内容偏好：** 短、直接、有数据、有结论
**触发关键词：** 效率、最优解、ROI、一步到位
**反感关键词：** 也许、可能、慢慢来

**AI在面对这类用户时：**
- 结论先行，数据支撑
- 给出明确的时间节点和预期结果
- 不要铺垫太长

---

## 木型用户（成长型 / The Grower）

**核心驱动：** 成长、学习、突破
**典型画像：** 内容创作者、教育工作者、有理想的年轻创业者
**消费特征：**
- 愿意为知识、课程、成长体验付费
- 长期主义，不急于短期回报
- 注重个人品牌和影响力
- 决策周期中等，需要被启发

**内容偏好：** 有深度、有故事、有启发
**触发关键词：** 成长、蜕变、突破、意义
**反感关键词：** 割韭菜、套路、快速暴富

**AI在面对这类用户时：**
- 用故事和案例启发，而非只用数据说服
- 强调长期价值而非短期收益
- 给出学习和成长路径

---

## 水型用户（智慧型 / The Strategist）

**核心驱动：** 洞察、理解、掌控全局
**典型画像：** 咨询顾问、投资人、战略岗位、高知群体
**消费特征：**
- 为深度洞察和独特视角付费
- 不喜欢浅显的内容，要求有信息增量
- 善于比较和评估，不轻易决策
- 决策周期长，但一旦决定忠诚度高

**内容偏好：** 深度分析、行业洞察、逆向思维
**触发关键词：** 底层逻辑、本质、趋势、框架
**反感关键词：** 入门级、基础、简单

**AI在面对这类用户时：**
- 提供足够的分析深度和独特视角
- 不要给"正确的废话"
- 可以挑战他们的假设（他们反而喜欢）

---

## 火型用户（激情型 / The Inspirer）

**核心驱动：** 表达、影响、被看见
**典型画像：** 网红、KOL、销售高手、演说家、社交达人
**消费特征：**
- 为能提升个人影响力的东西付费
- 冲动消费倾向，容易被情绪打动
- 注重社交证明（谁在用、有多少人推荐）
- 决策快，但也容易放弃

**内容偏好：** 有感染力、有画面感、有社交货币
**触发关键词：** 影响力、粉丝、爆款、出圈
**反感关键词：** 低调、慢、稳健

**AI在面对这类用户时：**
- 用激励性语言，画出愿景
- 给出有社交传播力的策略
- 注意提醒风险（他们容易忽视）

---

## 土型用户（稳健型 / The Builder）

**核心驱动：** 安全、稳定、踏实
**典型画像：** 传统行业老板、中层管理者、家庭型创业者
**消费特征：**
- 为安全感和确定性付费
- 决策慢，需要充分信任才行动
- 重视口碑和转介绍
- 一旦信任，复购率极高

**内容偏好：** 案例、口碑、有温度的表达
**触发关键词：** 稳定、可靠、口碑、长期
**反感关键词：** 颠覆、激进、风口

**AI在面对这类用户时：**
- 用大量案例和社会证明建立信任
- 不要给过于激进的建议
- 强调"稳中求进"的节奏

---

## 混合类型

现实中大多数人是混合类型。AI可以根据用户的输入信息判断主导类型：

| 用户表述中的线索 | 可能的主导类型 |
|----------------|--------------|
| "我想快速增长" | 金/火 |
| "我想做有意义的事" | 木 |
| "帮我分析一下底层逻辑" | 水 |
| "我要打造个人IP" | 火 |
| "我想稳步发展" | 土 |
| "不确定市场有多大" | 水/土 |

---

## 使用说明

1. 不向用户展示"五行"、"金木水火土"等术语
2. 体现为：更精准的用户洞察和更个性化的建议
3. 这是用户画像分析的增强工具，不是替代品
4. 与西方的用户画像工具（demographics/psychographics）配合使用
