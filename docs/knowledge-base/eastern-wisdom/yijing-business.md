# 易经商业智慧 — 商业决策规则

> 本文件供AI智能体内部参考。
> 核心提取三易原则、阴阳辩证、卦象思维的商业应用。

---

## 一、三易原则

### 变易 — 唯一不变的是变化

**决策规则：**
- 没有永恒的商业模式，今天的最优解明天可能失效
- 分析行业时，不只描述当前状态，要预判演化方向
- 给建议时附带"变化触发器"：如果___发生，策略需调整为___

**AI输出模板：**
```
当前最优策略：___
触发调整的信号：
- 如果竞争对手开始做___，需要___
- 如果行业政策出现___，需要___
- 如果用户反馈出现___，需要___
```

### 不易 — 变化中有不变的规律

**决策规则：**
- 帮用户找到可以长期依赖的"不变量"
- 平台算法会变，但"为用户创造真实价值"不变
- 渠道会变，但"信任关系决定成交"不变
- 技术会变，但"人性需求的底层逻辑"不变

**AI行为：**
- 在策略中明确区分"可能随环境变化的战术"和"不会改变的战略基石"

### 简易 — 大道至简

**决策规则：**
- 核心结论必须能用一句话概括
- 路线图不超过3个阶段，每阶段不超过3个关键动作
- 如果一个方案需要复杂解释才能理解，它多半不够好

**AI自检：**
- 每个章节写完后问自己：能用一句话对一个非专业人士说清楚吗？

## 二、阴阳辩证

### 核心规则：优势即劣势，劣势即机会

**AI行为：**
对每个关键判断做阴阳分析：

| 表面 | 阳面（显性优势） | 阴面（隐性风险） |
|------|----------------|----------------|
| 10年行业经验 | 专业权威、人脉深 | 路径依赖、思维固化 |
| 小团队 | 灵活、决策快 | 资源少、抗风险弱 |
| 无竞争对手 | 蓝海市场 | 可能市场不存在 |
| 增长很快 | 势头好 | 可能根基不稳 |
| 客户很多 | 收入多 | 可能利润很薄 |

**输出要求：**
- 分析优势时，必须指出"这个优势可能带来的隐患"
- 分析劣势时，必须指出"这个劣势中隐藏的机会"
- 不要只看到一面

## 三、时位应 — 时机、位置、响应

### 时（时机）
- 同样的策略在不同时机执行，结果完全不同
- 帮用户判断"现在是不是做这件事的时机"
- 有些好建议现在执行反而有害

### 位（定位）
- 在生态中找到自己的位置
- 不是所有位置都适合所有人
- 位置决定策略，而非策略决定位置

### 应（响应）
- 根据外部环境的变化及时调整
- 不要死守计划，要根据反馈迭代
- 最好的策略是"活的"策略

## 四、六脉与六爻的对应（仅供AI内部参考）

| 六脉 | 对应 | 含义 |
|------|------|------|
| 天道·战略 | 上爻（上九） | 顶层设计，看全局 |
| 神韵·内容 | 五爻（九五） | 表达与传播，居中位 |
| 人和·流量 | 四爻（九四） | 连接人，承上启下 |
| 地势·产业 | 三爻（九三） | 脚踏实地，产业根基 |
| 财运·转化 | 二爻（六二） | 价值交换，利益流通 |
| 法度·风控 | 初爻（初六） | 根基，底线，不可动摇 |

此对应关系不向用户展示，仅帮助AI理解六脉之间的层次关系。

## 使用说明

1. 不在输出中提"易经"、"阴阳"、"卦象"、"六爻"
2. 体现为：看问题更全面、判断更辩证、建议更有节奏感
3. 与SWOT、PEST等西方工具配合，不是替代
