# 六脉蜂群 - 最终定义（v3.0）

> 2026年3月2日 - 基于实战的商业洞察
> 每一脉解决一个核心问题

---

## 一、六脉终极定义

```
1. 天道（定位）：做不做？（方向选择）
2. 地利（产品）：卖什么？（产品定义）
3. 人和（模式）：怎么连？（生态位）
4. 神韵（内容）：怎么说？（认知占领）
5. 财帛（转化）：怎么收钱？（变现路径）
6. 法度（风险）：如何不死？（生存底线）
```

---

## 二、为什么这样定义？

### 2.1 天道（定位）：做不做？

**核心问题：这个生意值得做吗？**

```
创业者最怕的：选错赛道，一年白干

天道·战略Agent回答：
- 市场规模有多大？
- 竞争是否红海？
- 我有没有优势？
- 赚钱的概率大吗？

输出：GO / NO-GO 决策 + 如果做，怎么定位
```

**例子：**
```
用户问："美业培训能做吗？"

天道分析：
❌ 红海竞争，差异化难
✅ 细分市场（创业者教育）有蓝海机会
✅ 你的培训经验是壁垒
✅ 结论：可以做，但定位"创业者教育"
```

---

### 2.2 地利（产品）：卖什么？

**核心问题：我卖什么产品/服务？**

```
选对赛道后，接下来定义产品

地利·产品Agent回答：
- 核心产品是什么？（单一爆品）
- 产品矩阵怎么设计？（如何扩展）
- 产品护城河在哪？（如何形成壁垒）

核心逻辑：产品足够长，就是产业
```

**例子：**
```
用户问："我的核心产品应该是什么？"

地利分析：
❌ 不要一开始就想做全产业链
✅ 先做一个"爆品"：美业创业者实战课
✅ 第二步：延伸到"工具化"：培训SaaS
✅ 第三步：扩展到"生态化"：供应链平台

产品进化路径：单一爆品 → 产品线 → 产业链
```

---

### 2.3 人和（模式）：怎么连？

**核心问题：如何设计生态位？**

```
有产品了，但一个人打不了天下

人和·模式Agent回答：
- 谁是我的盟友？（上游供应商、下游渠道）
- 谁是我的客户？（C端、B端、政府）
- 谁是我的对手？（直接竞争、间接竞争）
- 如何团结更多人？（建立生态）

核心逻辑：商业模式 = 生态设计能力
```

**例子：**
```
用户问："美业培训如何获客？"

人和分析：
❌ 不要只想着投放广告（成本高、效果差）
✅ 盟友：培训机构（分成：30%-70%）
✅ 盟友：企业内训（B端大客户）
✅ 客户：想转行的美业从业者（精准需求）
✅ 生态：打造"美业创业者联盟"

核心：不是单打独斗，而是生态协同
```

---

### 2.4 神韵（内容）：怎么说？

**核心问题：如何占领用户认知？**

```
有产品、有模式，但用户不知道你

神韵·内容Agent回答：
- 你的品牌人设是什么？（专家/朋友/导师？）
- 你的内容风格是什么？（专业/亲和/搞笑？）
- 在哪个平台发声？（小红书/抖音/公众号）
- 什么内容能火？（痛点/干货/故事？）

核心逻辑：内容时代，内容即流量
```

**例子：**
```
用户问："如何让更多人知道我的美业培训？"

神韵分析：
❌ 不要硬广（平台会限流）
✅ 人设："10年美业老兵，帮助100+创业者成功"
✅ 平台：小红书（图文）+ 抖音（视频）
✅ 内容：
  - 干货：美业避坑指南
  - 故事：学员转型案例
  - 金句："美业不是技术生意，是人心生意"
✅ 策略：日更（3-5条）→ 积累粉丝 → 转化变现

核心：内容 = 免费、精准、持久的流量
```

---

### 2.5 财帛（转化）：怎么收钱？

**核心问题：如何设计变现路径？**

```
有流量了，但流量不等于钱

财帛·转化Agent回答：
- 变现路径怎么设计？（免费→低价→中价→高价）
- 定价策略是什么？（成本/价值/竞争）
- 转化漏斗如何优化？（认知→兴趣→付费→复购）
- LTV（客户终身价值）如何最大化？

核心逻辑：不是一次交易，而是长期关系
```

**例子：**
```
用户问："美业培训怎么收费？"

财帛分析：
❌ 不要只卖高价课（转化率低）
✅ 设计变现路径：

  免费层：小红书干货内容（获客）
    ↓ 3% 转化
  低价层：99元体验课（尝鲜）
    ↓ 15% 转化
  中价层：2990元系统课（主力）
    ↓ 30% 转化
  高价层：19800元私教（高客单）

核心：降低门槛，提高复购，最大化LTV
```

---

### 2.6 法度（风险）：如何不死？

**核心问题：如何避免"翻车"？**

```
生意做大了，但风险也在增加

法度·风险Agent回答：
- 哪些词不能用？（广告法违禁词）
- 哪些承诺不能做？（法律风险）
- 哪些平台规则要注意？（封号风险）
- 哪些钱不能赚？（道德底线）

核心逻辑：活下来，比做大更重要
```

**例子：**
```
用户问："美业培训如何避免风险？"

法度分析：
❌ 不能承诺："包就业"、"月入10万"（虚假宣传）
❌ 不能说："行业第一"、"最专业"（绝对化用语）
✅ 建议说："已帮助100+学员成功转型"（可验证）
✅ 必须避坑：
  - 广告法：不用"第一"、"最"、"顶级"
  - 平台规则：不引导私聊、不直接留微信
  - 合同风险：明确退款政策、不夸大效果

核心：合规不是束缚，是护城河
```

## 四、实战案例完整版

### 案例：美业培训创业者

```
创业者：王姐，10年美业经验，想做培训生意

[天道·战略] 做不做？
分析：美业培训是红海，但"创业者教育"是蓝海
      你的10年经验是壁垒
      结论：可以做，定位"创业者实战导师"

[地利·产品] 卖什么？
核心产品：《美业创业实战课》（12周线上+线下）
产品矩阵：
  - 单一爆品：实战课（2990元）
  - 工具化：学员SaaS系统（99元/月）
  - 生态化：美业供应链平台（未来）

[人和·模式] 怎么连？
盟友：
  - 合作培训机构（提供场地，分成30-70%）
  - 美业博主（推荐学员，佣金20%）
客户：
  - 想转行的美业从业者（精准）
  - 小型美业店主（B端）
对手：
  - 泛培训课程（差异化：实战 vs 理论）
  - 在线学习平台（差异化：陪伴 vs 一次性）

[神韵·内容] 怎么说？
品牌人设："10年美业老兵，帮助100+学员成功"
内容矩阵：
  - 小红书：创业故事+避坑指南（日更）
  - 抖音：培训现场+学员访谈（周更）
  - 公众号：深度方法论（月更）
流量策略：
  - 自然流量：SEO + 平台推荐
  - 社交裂变：学员口碑传播
  - 付费放大： occasional 投放（ROI > 3）

[财帛·转化] 怎么收钱？
变现路径：
  - 免费：小红书内容（获客）
  - 引流：99元直播课（转化率3%）
  - 主力：2990元系统课（转化率15%）
  - 高价：19800元私教（转化率5%）
  - 持续：99元/月 SaaS工具（复购率40%）
  - B端：企业内训（客单价5万+）

[法度·风险] 如何不死？
合规清单：
  - 广告法：不用"第一"、"最"、"保证"
  - 平台规则：不引导私聊、不直接留微信
  - 法律风险：明确退款政策、不夸大效果
  - 道德底线：不收割小白、不卖焦虑

数据飞轮：
  用户反馈 → 持续优化课程 → 口碑更好 → 更多用户
```

## 五、六脉的相互关系

### 5.1 依赖关系

```
天道是"大脑"：决策做不做
地利依赖天道：有了定位，才能定义产品
人和依赖地利+天道：有了产品，才能设计生态
神韵依赖地利+天道+人和：有了产品和模式，才能生产内容
财帛依赖人和+神韵：有了模式和流量，才能设计变现
法度依赖所有：所有环节都要合规
```

### 5.2 数据流

```
天道输出：战略定位
  ↓ 地利输入
地利输出：产品定义
  ↓ 人和输入
人和输出：商业模式
  ↓ 神韵输入
神韵输出：品牌内容
  ↓ 财帛输入
财帛输出：变现路径
  ↓ 所有环节输入
法度输出：风险清单
```
