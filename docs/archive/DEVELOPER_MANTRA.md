## 🐝 蜂群思维（核心认知）

```
不是六个工具，而是一个智能蜂群
- 每个Agent有独立智能
- Agent之间可以通信
- Agent可以协同作战
- 蜂群涌现出超越个体的智慧
```

**蜂群拓扑（2026.03重新定义）：**
```
        Queen Agent（任务分解、协调）
                 │
    ┌────────────┼────────────┐
    ↓            ↓            ↓
天道·战略  神韵·内容  地利·产品
    ↓            ↓            ↓
人和·模式  财帛·转化  法度·风险
```

**新六脉核心逻辑（2026.03最终版）：**
- **天道（定位）**：做不做？（方向选择）
- **地利（产品）**：卖什么？（产品定义）
- **人和（模式）**：怎么连？（生态位）
- **神韵（内容）**：怎么说？（认知占领）
- **财帛（转化）**：怎么收钱？（变现路径）
- **法度（风险）**：如何不死？（生存底线）

**每一脉的输出，都是下一脉的输入，且Agent可以点对点通信。**

---

## 🎨 设计铁律

```
干净 > 花哨
功能 > 美观
一致 > 个性
```

**禁止事项：**
- ❌ 装饰性 SVG、曲线、渐变
- ❌ emoji 作为 UI 元素
- ❌ font-bold, text-xl+
- ❌ framer-motion
- ❌ opacity-0 的交互元素

---

## 💻 代码原则

```
1. 一文件不超过 500 行
2. 先让一个模块完美，再抽象
3. 类型严格（noImplicitAny）
4. 永远设置 timeout
5. 永远验证输入
6. 永远优雅降级
```

---

## 📐 布局标准

**三栏布局：**
```
[左侧 280px] | [中间 400px] | [右侧 flex-1]
   历史          输入           输出
```

**折叠按钮：**
```tsx
w-6 h-12 rounded-full shadow-md
hover:bg-indigo-50 hover:border-indigo-300
```

**颜色系统：**
- 暗色主背景：#0A0A0A → #111111 → #161616
- 强调色：indigo-600
- 文字：slate-300 / slate-500
